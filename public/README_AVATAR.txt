# Public Static Files Folder

You can manually upload your own 3D avatar `.glb` or `.gltf` files here!

## How to use:
1. Upload your custom `.glb` model into this `public` folder (for example, name it `my-avatar.glb`).
2. Once the file is here, you can reference it in the application or in `/src/presets.ts` using the local URL path:
   `/my-avatar.glb`

By default, any file in the `public` folder is served from the root URL of your website.
